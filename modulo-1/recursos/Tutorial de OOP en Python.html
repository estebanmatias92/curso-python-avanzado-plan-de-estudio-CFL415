Tutorial de OOP en Python

Clases vs Objetos, Metodos vs Propiedades

En Python, una clase se usa para definir un tipo de objeto, especificando las propiedades y comportamientos (metodos) que deberían tener los objetos de ese tipo. Un objeto es una instancia de una clase.

Las propiedades (o atributos) son las variables que pertenecen a un objeto, y los métodos son las funciones que pertenecen a un objeto.


    class Coche:
        color = "rojo"  # propiedad de la clase
        def arrancar(self):  # método de la clase
            print("El coche ha arrancado")

    mi_coche = Coche()  # creando un objeto de la clase Coche
    print(mi_coche.color)  # accediendo a una propiedad del objeto
    mi_coche.arrancar()  # llamando a un método del objeto

Encapsulamiento (Modificadores de Acceso)

El encapsulamiento implica hacer privados los atributos y métodos de una clase para prevenir modificaciones no autorizadas.


    class Coche:
        def __init__(self):
            self.__color = "rojo"  # atributo privado
    
        def obtener_color(self):  # getter
            return self.__color
    
        def establecer_color(self, color):  # setter
            self.__color = color
    
    mi_coche = Coche()
    print(mi_coche.obtener_color())  # rojo
    mi_coche.establecer_color("azul")
    print(mi_coche.obtener_color())  # azul

Herencia

La herencia permite definir una clase que hereda todos los métodos y propiedades de otra clase.


    class Vehiculo:
        def arrancar(self):
            print("El vehículo ha arrancado")
    
    class Coche(Vehiculo):  # Coche hereda de Vehiculo
        pass
    
    mi_coche = Coche()
    mi_coche.arrancar()  # El vehículo ha arrancado

Polimorfismo

El polimorfismo permite usar una interfaz única para representar diferentes tipos de datos.


    class Coche:
        def arrancar(self):
            print("El coche ha arrancado")
    
    class Moto:
        def arrancar(self):
            print("La moto ha arrancado")
    
    def arrancar_vehiculo(vehiculo):
        vehiculo.arrancar()
    
    mi_coche = Coche()
    mi_moto = Moto()
    arrancar_vehiculo(mi_coche)  # El coche ha arrancado
    arrancar_vehiculo(mi_moto)  # La moto ha arrancado

Clases Abstractas (Interfaces)

Una clase abstracta es una clase que no puede ser instanciada y sirve como una interfaz para otras clases.


    from abc import ABC, abstractmethod
    
    class Animal(ABC):
        @abstractmethod
        def hacer_sonido(self):
            pass
    
    class Perro(Animal):
        def hacer_sonido(self):
            print("Guau")
    
    mi_perro = Perro()
    mi_perro.hacer_sonido()  # Guau
